How 2 start a blog with Hexo
Created 2020-07-15| Updated 2020-07-21
Post View:


1.准备工作：

1、git

2、Nodejs

3、hexo


2.部署git

yum intall -y git


可以使用git –version查看版本并确认安装是否成功

配置账号：

git config --global user.name "git用户名"

git config --global user.email "git邮箱"

ssh-keygen -t rsa -C "git邮箱"

生成公钥的目录可在提示信息中查看
配置完成后可从GitHub的个人设置中设置SSH密钥，将id_rsa.pub文件中所有的内容复制到对应地方即可

3.配置nodejs

wget https://npm.taobao.org/mirrors/node/v11.0.0/node-v11.0.0.tar.gz

解压安装包

tar -xzvf node-v11.0.0.tar.gz

进入目录并安装相关插件

cd node-v11.0.0

sudo yum install gcc gcc-c++

进行默认配置并编译

./configure

make

编译完之后就可以开始安装

sudo make install

之后验证安装

node -v

检测安装是否完成:参考文献

4.部署hexo

npm install -g hexo-cli

安装完成后依旧可以通过hexo -v查看版本

通过mkdir在你喜欢的位置建立文件夹，例如:/usr/local/blog

进入blog文件夹，初始化hexo

hexo init （若权限不够，使用sudo即可）

当提示Start blogging with Hexo!则表示初始化完成

启动博客：hexo s

访问localhost:4000，博客默认在localhost的4000端口启动，一般做预览

新建一篇文章

hexo n "xxxxxx"

博客具体内容在markdown文档内进行修改即可

退回blog目录后

hexo clean

hexo g

hexo s

刷新下localhost：4000页面，可见之前写的静态文章已经发布上去了

5.在GitHub上发布内容：
参考链接

6.更新内容

hexo g

hexo d

输入你的github账号密码稍等即可
